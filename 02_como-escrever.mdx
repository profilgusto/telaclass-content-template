# Briefing

Como dito, o MDX é uma mistura de Markdown e JSX. Então, é possivel escrever seus documentos (como este daqui que vos fala) utilizando uma mistura das sintaxes de JSX e Markdown. 


# Cabeçalhos

Como já deve ter notado, cabeçalhos são feitos utilizando o hashtag. Há três níveis.

Exemplo: 

\# Heading 1

\#\# Heading 2

\#\#\# Heading 3

Note que os exemplos não se tornaram headers de fato pois o hashtag foi escapado com o uso da "\\". Você pode fazer isso para qualquer escapar qualque caractere especial.

# E aí?

OK, até então estilizamos o conteúdo usando somente a sintaxe de *Markdown*. Porém, aqui estamos em um ambiente mais poderoso, o **Markdown Extended**!!!

Em um nível básico, podemos também estilizar os componentes através de tags JSX:

<p>Este é um texto comum usando tag JSX.</p>

Esta é uma lista usando JSX:
<ul>
    <li>Item 1</li>
    <li>Item 2</li>
    <li>**Item 3** (no que pode-se misturar os dois!)</li>
</ul>

E por aí vai. 

Naturalmente, prefira estilizar o básico usando as tags de Markdown. O poder aqui está realmente em poder usar códigos javascript, definir e usar componentes, importar classes css, etc., tudo misturado com o Markdown!!!


# Expressões

Pode-se usar expressões em JS dentro de \{ \<expressão \>\}:

- 90 graus = {Math.PI / 2} radianos.
- 180 graus = {Math.PI} radianos.
- 270 graus = {3*Math.PI/2} radianos.
- 360 graus = {2*Math.PI} radianos.

Você pode escrever literalmente qualquer coisa em JS, desde que esteja dentro de \{\<code\>\}, e que seja uma função que retorna algo renderizável:

Abaixo está uma função que renderiza um texto diferente de acordo com um valor sorteado aleatoriamente.

{ (function () {
        const chute = Math.random()

        if (chute > 0.66) {
            return <span style={{color: 'tomato'}}> Olha aqui! </span>
        }

        if (chute > 0.3) {
            return <span style={{color: 'violet'}}> Quem teria adivinhado!? </span>
        }

        return <span style={{color: 'goldenrod'}}> Não sou eu. </span>
    })
()}

# Matemática

Através de um plugin habilitado, é possível escrever equações em Latex dentro de ambientes matemáticos como se estivesse realmente no próprio latex: `$ <equação> $`.

Para escrever uma equação inline: $c = 2 \pi r$.


Também é possível escrever uma equação em nova linha:
$$
c = \int_a^b x^2 dx
$$

Se precisar, pode expandir as funcionalidades deste ambiente matemático utilizando o [remark-mdx-math-enhanced](https://github.com/goodproblems/remark-mdx-math-enhanced).
